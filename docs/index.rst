===================================
Welcome to the ik_lib.h documentation!
===================================

**ik_lib** is a library that aims to optimize and simplify the C/C++ 
coding experience by automating certain tasks, providing helpful functions
and (re)defining certain types

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.


.. note::

   This project is under active development.
